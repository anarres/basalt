Welcome to Basalt

Basalt is a simple, flexible, rock-solid HTML and CSS starter theme.

It works in all the major browsers, on all screen resolutions, and can gracefully handle any valid HTML that gets dumped into it.

It's a two-column theme. The columns are called 'main' and 'sidebar'. The main column should come first in the HTML code so that on smaller devices it will float above the sidebar (since the information in 'main' should be more important than the information in 'sidebar'). So if you want the sidebar on the right instead of on the left don't touch the HTML, just modify basalt.css to switch the floats on the main and sidebar divs.

Basalt was created by Foofurple. It is dedicated to the Public Domain and you are welcome to use it in any way you wish.
